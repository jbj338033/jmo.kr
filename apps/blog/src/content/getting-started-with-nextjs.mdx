Next.js는 React 기반의 풀스택 프레임워크입니다.

## 왜 Next.js인가?

1. **서버 컴포넌트** - 성능 최적화
2. **파일 기반 라우팅** - 직관적인 구조
3. **이미지 최적화** - 자동 최적화
4. **Turbopack** - 빠른 개발 경험

## 프로젝트 생성

```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```

## App Router

Next.js 13부터 도입된 App Router는 React Server Components를 기본으로 사용합니다.

```
app/
├── layout.tsx    # 루트 레이아웃
├── page.tsx      # 홈페이지
└── posts/
    └── [slug]/
        └── page.tsx  # 동적 라우트
```
